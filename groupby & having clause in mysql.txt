GROUP BY
-----
group rows which have same values into summary rows

EG:SELECT COUNT(STUDENTID),ADDRESS FROM TEST1.STUDENT 
GROUP BY ADDRESS ;


HAVING
----
always be used along with aggregate functions when the criteria is met and it is similar to where clause

EG :SELECT COUNT(STUDENTID),ADDRESS FROM TEST1.STUDENT 
GROUP BY ADDRESS HAVING COUNT(STUDENTID)>1 ;

SYNTAX FOR GROUP BY,ORDER BY AND HAVING
-----
SELECT COLUMNAME1,COLUMNANME2 FROM TABLENAME WHERE CONDITION 
GROUP BY COLUMNAME 
ORDER BY COLUMNAME
HAVING CONDITION;
