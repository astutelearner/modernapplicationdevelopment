JOINS IN MYSQL
-----
1.INNER JOIN - Intersection
2.LEFT (OUTER JOIN) - return all records from left table and only matched records from right table
3.RIGHT (OUTER JOIN) - return all records from right table and only matched recors from left table
4.FULL OUTER JOIN - return all records when there is a match in either left or right tables

Syntax of inner join
----
SELECT * FROM TABLE1 INNER JOIN TABLE2 ON TABLE1.MATCHINGCOLUMN=TABLE2.MATCHINGCOLUMN

(OR)

SELECT * FROM TABLE1 JOIN TABLE2 ON TABLE1.MATCHINGCOLUMN=TABLE2.MATCHINGCOLUMN

EG: SELECT * FROM TEST1.DEPARTMENT JOIN TEST1.FACULTY ON DEPARTMENT.DEPARTMENTID=FACULTY.DEPARTMENTID;

Syntax of left join
-----
SELECT * FROM TABLE1 LEFT JOIN TABLE2 ON TABLE1.MATCHINGCOLUMN=TABLE2.MATCHINGCOLUMN

EG: SELECT * FROM TEST1.DEPARTMENT LEFT JOIN TEST1.FACULTY ON DEPARTMENT.DEPARTMENTID=FACULTY.DEPARTMENTID;

Syntax of right join
--------------
SELECT * FROM TABLE1 RIGHT JOIN TABLE2 ON TABLE1.MATCHINGCOLUMN=TABLE2.MATCHINGCOLUMN;

SELECT * FROM TEST1.DEPARTMENT RIGHT JOIN TEST1.FACULTY ON DEPARTMENT.DEPARTMENTID=FACULTY.DEPARTMENTID;

Syntax of full join
------
SELECT * FROM TABLE1 FULL JOIN TABLE 2

eg: SELECT * FROM TEST1.DEPARTMENT FULL JOIN TEST1.FACULTY;
